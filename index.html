The following code examples were created while taking Udacity's JavasScript Design Patterns course.

The Cat clicker is an app that displays photos of various cats. Users can click on the photo of cats and cast votes.

Here are the different versions of the app:

  1. Version 1 - 1 Cat is hard coded on the page.
  2. Version 1 - 5 cats are dynamically loaded on to page.
  3. Version 2 - 5 cats are dynamically loaded on to page. JS is in a separate file.
  4. Version 2b - 5 cats are dynamically loaded on to page. A user must select the cat to view.
  5. Version 2c - 5 cats are dynamically loaded on to page. A user must select the cat to view. My code is modified to match the solution to the lesson.
  6. Version 3 - KnockoutJS is being used in this version.
  7. Final Project
